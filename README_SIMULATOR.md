# Football Game Statistics Simulator 🎮⚽

A fully interactive web app that simulates football (soccer) matches with realistic statistics and detailed analytics.

## 🌐 LIVE LINK - PLAY NOW!

### **[⚽ CLICK HERE TO PLAY: Football Game Simulator](https://football-game-simulator.streamlit.app/)**

## Features

✅ **Interactive Game Simulation**
- Simulates realistic football matches with 100+ configurable events
- Two teams competing with physics-based mechanics
- Player skill ratings affect outcomes

✅ **Comprehensive Statistics**
- Individual player stats (goals, assists, shots, passes, tackles, etc.)
- Team statistics (possession %, pass accuracy, defensive metrics)
- Real-time score tracking

✅ **Advanced Analytics**
- Interactive charts and visualizations (Plotly)
- Team comparison metrics
- Possession distribution
- Player performance breakdown

✅ **Customization**
- Customize team names and formations
- Adjust number of game events
- Set random seed for reproducible results
- Pre-built rosters with famous players

✅ **Mobile-Friendly**
- Responsive design works on all devices
- Touch-friendly interface
- Optimized for Play Store deployment
✅ **Play Store Ready**
- Connectable API architecture
- Native app wrapper compatible (React Native/Flutter)
- Cloud deployment ready

## How to Play

1. **Visit the Live App:** https://football-game-simulator.streamlit.app/
2. **Configure Teams** - Set team names and formations in the sidebar
3. **Set Game Events** - Choose number of events (10-500)
4. **Start Simulation** - Click "⚽ START SIMULATION"
5. **View Results** - Browse through 5 tabs of detailed statistics


## How the Simulation Works

1. **Event Generation** - 100+ random events occur during the match
2. **Event Types** - Pass, Shot, Tackle, Foul, Interception
3. **Skill-Based Outcome** - Player attributes (offense, defense, stamina) affect results
4. **Weighted Probability** - Events are weighted realistically
5. **Statistics Tracking** - All actions recorded for comprehensive analytics
